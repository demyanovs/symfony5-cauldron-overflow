# Symfony5 - Cauldron overflow

### Install symfony cli
``
wget https://get.symfony.com/cli/installer -O - | bash
``  

### To start a web server  
``
symfony serve -d
``  
or  
``
php -S 127.0.0.1:8000 -t public/
``
